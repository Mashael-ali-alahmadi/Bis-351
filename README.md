# GlowCare Skincare

## Project Description
GlowCare Skincare is a website that helps users learn about skincare routines, knowing the proper product, and read helpful skincare tips.

## The Problem
Some people want to take care of their skin but don’t know their skin type or how to use the best skin care routine. This may cause a  skin problems because they may end up using the wrong products.
